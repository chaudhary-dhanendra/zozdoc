# Benchmarks

Benchmarks define measurable evidence for HermesNet performance and certification.

## Latency Benchmarks

Measure end-to-end and component latency with documented environment, workload, and percentile reporting.

## Replay Benchmarks

Measure deterministic replay throughput, recovery time, and output equivalence.

## Matching Benchmarks

Measure matching-engine throughput, contention behavior, and deterministic order handling.

## Memory Benchmarks

Measure allocation, peak memory, snapshot size, and recovery memory profile.

## Certification Benchmarks

Certification benchmarks must be reproducible, versioned, and tied to release evidence.
