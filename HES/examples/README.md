# Examples

This directory is reserved for non-normative engineering examples that support implementation and testing.

## Reference Implementations

Reference implementations illustrate specified behavior but do not override HES normative text.

## Sample EngineEvents

Sample `EngineEvent` streams may be used for replay, conformance, and documentation examples.

## Sample Snapshots

Snapshots should identify schema version, source specification version, and compatibility notes.

## Sample Journals

Journals should preserve deterministic ordering and include enough metadata for replay validation.

## Golden Vectors

Golden vectors are canonical input/output pairs used for conformance and regression testing.
